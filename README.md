# Resumo-pdf-ia
Sistema inteligente de análise de PDFs integrado à IA. Permite enviar múltiplos PDFs, extrair texto, gerar resumos automáticos, responder perguntas baseadas no conteúdo e baixar os resultados em TXT ou DOCX. Construído com Python, Gradio, OpenAI e PyPDF2. Focado em automação, produtividade e demonstração de integração real com IA.
